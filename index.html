Sobre mí

Soy desarrolladora web y móvil con formación en arquitectura, apasionada por crear soluciones tecnológicas innovadoras que mejoren la vida cotidiana de las personas. Me encanta combinar mi visión creativa con habilidades técnicas para diseñar y desarrollar proyectos funcionales y atractivos. Tengo experiencia trabajando con HTML, CSS, JavaScript, Bootstrap, Java y Spring Boot, creando interfaces intuitivas y eficientes. También manejo Figma para prototipos y Android Studio para aplicaciones móviles. Mi enfoque proactivo, junto con mi conocimiento en Scrum, me permite gestionar proyectos de manera ágil y efectiva. Siempre busco aprender nuevas tecnologías que aporten valor y generen un impacto positivo en la sociedad.

Mis proyectos

SOLAR SPORTS
GitHub

SOLAR Sports es una aplicación móvil, desarrollada con Android Studio y Java, que proporciona información personalizada para la instalación de paneles solares en espacios deportivos. La app permite calcular la viabilidad del proyecto, visualizar estadísticas energéticas, facilitar la venta de energía y acceder a artículos actualizados sobre avances en energía solar.

INTI SUSTAINABLE MARKET
GitHub

INTI es un e-commerce de productos sostenibles, desarrollado con un enfoque académico utilizando Java, HTML, Bootstrap y Spring Boot. La plataforma permite a los usuarios explorar y adquirir productos ecológicos, promoviendo un consumo responsable. Además, cuenta con una interfaz intuitiva, un sistema de gestión de pedidos eficiente y herramientas para fomentar el comercio sostenible

VEGANXPRESS FOOD
GitHub

Vegan Xpress Food es una aplicación web, desarrollada con un enfoque académico utilizando Express.js, MySQL, Docker, Vue.js y Bootstrap, que permite a los usuarios ver el menú, hacer pedidos y reservaciones. Además, cuenta con un panel de administración donde los usuarios pueden gestionar el contenido del menú, modificar ítems y supervisar pedidos, optimizando la experiencia del restaurante.

RetoFit
GitHub

RetoFit es una landing page desarrollada durante una competencia de hackathon en Generation Colombia, utilizando HTML, JavaScript y Bootstrap. La plataforma gamifica la actividad física, incentivando a los usuarios a realizar ejercicio de manera divertida y motivadora a través de desafíos y recompensas.
